Formas de aplicar estilos CSS

Existen tres formas de aplicar estilos CSS a un documento HTML.

  1. Estilos en línea.
  2. Estilos internos.
  3. Estilos externos.

1. Estilos en línea

Se aplican directamente a los elementos HTML mediante el atributo <style>. No es recomendable para proyectos grandes debido a su poca mantenibilidad.

<p style="font-size: 20px; color: darkorange;">Este es un párrafo con estilo en línea.</p>

Este es un párrafo con estilo en línea.

2. Estilos internos

Estos estilos se colocan dentro de la etiqueta <style> en la sección <head> del HTML.


  <head>
    <style>
      body {
        background: black;
        color: white;
      }
      h1 {
        font-size: 36px;
        color: orange;
      }
    </style>
  </head>

3. Estilos externos

Es el método más eficiente y recomendado. Los estilos se almacenan en un archivo CSS separado que se enlaza al archivo HTML a través de la etiqueta <link>.

  
  <head>
    <link rel="stylesheet" href="style.css">
  </head>
